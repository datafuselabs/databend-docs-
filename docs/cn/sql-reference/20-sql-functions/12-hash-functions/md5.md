---
title: MD5
---

计算字符串的MD5 128位校验和。返回值为32个十六进制字符的字符串，如果参数为NULL，则返回NULL。

## 语法

```sql
MD5(<expr>)
```

## 示例

```sql
SELECT MD5('1234567890');

┌──────────────────────────────────┐
│         md5('1234567890')        │
├──────────────────────────────────┤
│ e807f1fcf82d132f9bb018ca6738a19f │
└──────────────────────────────────┘
```
