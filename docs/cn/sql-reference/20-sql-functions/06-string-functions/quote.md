---
title: QUOTE
---

对字符串进行引用，以生成可在 SQL 语句中用作正确转义数据值的结果。

## 语法

```sql
QUOTE(<str>)
```

## 示例

```sql
SELECT QUOTE('Don\'t!');
+-----------------+
| QUOTE('Don't!') |
+-----------------+
| Don\'t!         |
+-----------------+

SELECT QUOTE(NULL);
+-------------+
| QUOTE(NULL) |
+-------------+
|        NULL |
+-------------+
```
