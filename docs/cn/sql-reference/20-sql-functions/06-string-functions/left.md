---
title: LEFT
---

返回字符串 str 的最左边 len 个字符，如果任何参数为 NULL，则返回 NULL。

## 语法

```sql
LEFT(<str>, <len>);
```

## 参数

| 参数      | 描述                                              |
|-----------|----------------------------------------------------------|
| `<str>`   | 要从中提取字符的主字符串 |
| `<len>`   | 字符的数量                                  |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT LEFT('foobarbar', 5);
+----------------------+
| LEFT('foobarbar', 5) |
+----------------------+
| fooba                |
+----------------------+
```
