---
title: RIGHT
---

返回字符串 str 的最右边 len 个字符，如果任何参数为 NULL，则返回 NULL。

## 语法

```sql
RIGHT(<str>, <len>);
```

## 参数

| 参数      | 描述                                              |
|-----------|--------------------------------------------------|
| `<str>`   | 要从中提取字符的主字符串                         |
| `<len>`   | 字符的数量                                       |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT RIGHT('foobarbar', 4);
+-----------------------+
| RIGHT('foobarbar', 4) |
+-----------------------+
| rbar                  |
+-----------------------+
```
