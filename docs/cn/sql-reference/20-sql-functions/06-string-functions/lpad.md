---
title: LPAD
---

返回字符串 `str`，左侧填充字符串 `padstr` 至长度为 `len` 字符。
如果 `str` 的长度超过 `len`，则返回值会被截断为 `len` 字符。

## 语法

```sql
LPAD(<str>, <len>, <padstr>)
```

## 参数

| 参数       | 描述         |
|------------|--------------|
| `<str>`    | 字符串。     |
| `<len>`    | 长度。       |
| `<padstr>` | 填充字符串。 |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT LPAD('hi',4,'??');
+---------------------+
| LPAD('hi', 4, '??') |
+---------------------+
| ??hi                |
+---------------------+

SELECT LPAD('hi',1,'??');
+---------------------+
| LPAD('hi', 1, '??') |
+---------------------+
| h                   |
+---------------------+
```
