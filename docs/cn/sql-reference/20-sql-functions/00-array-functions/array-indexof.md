---
title: ARRAY_INDEXOF
---

如果数组中包含某个元素，则返回该元素的索引（基于1）。

## 语法

```sql
ARRAY_INDEXOF( <array>, <element> )
```

## 示例

```sql
SELECT ARRAY_INDEXOF([1, 2, 9], 9);

┌─────────────────────────────┐
│ array_indexof([1, 2, 9], 9) │
├─────────────────────────────┤
│                           3 │
└─────────────────────────────┘
```
