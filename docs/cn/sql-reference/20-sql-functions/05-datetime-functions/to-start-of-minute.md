---
title: TO_START_OF_MINUTE
---

将带有时间的日期（timestamp/datetime）向下舍入到分钟的开始。

## 语法

```sql
TO_START_OF_MINUTE( <expr> )
```

## 参数

| 参数      | 描述       |
|-----------|------------|
| `<expr>`  | timestamp  |

## 返回类型

`TIMESTAMP`，返回格式为“YYYY-MM-DD hh:mm:ss.ffffff”的日期。

## 示例

```sql
SELECT
  to_start_of_minute('2023-11-12 09:38:18.165575');

┌──────────────────────────────────────────────────┐
│ to_start_of_minute('2023-11-12 09:38:18.165575') │
├──────────────────────────────────────────────────┤
│ 2023-11-12 09:38:00                              │
└──────────────────────────────────────────────────┘
```
