---
title: TO_SECOND
---

将带有时间的日期（timestamp/datetime）转换为包含分钟中秒数（0-59）的 UInt8 数字。

## 语法

```sql
TO_SECOND(<expr>)
```

## 参数

| 参数      | 描述       |
|-----------|------------|
| `<expr>`  | 时间戳     |

## 返回类型

`TINYINT`

## 示例

```sql
SELECT
    to_second('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ to_second('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│                                      18 │
└─────────────────────────────────────────┘
```
