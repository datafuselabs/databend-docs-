---
title: TO_DAY_OF_YEAR
---

将日期或带时间的日期（timestamp/datetime）转换为包含一年中第几天（1-366）的 UInt16 数字。

## 语法

```sql
TO_DAY_OF_YEAR(<expr>)
```

## 参数

| 参数       | 描述         |
| ---------- | ------------ |
| `<expr>`   | 日期/时间戳  |

## 返回类型

`SMALLINT`

## 示例

```sql
SELECT
    to_day_of_year('2023-11-12 09:38:18.165575');

┌──────────────────────────────────────────────┐
│ to_day_of_year('2023-11-12 09:38:18.165575') │
├──────────────────────────────────────────────┤
│                                          316 │
└──────────────────────────────────────────────┘
```
