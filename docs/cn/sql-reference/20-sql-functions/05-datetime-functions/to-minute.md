---
title: TO_MINUTE
---

将带有时间的日期（timestamp/datetime）转换为包含小时中分钟数（0-59）的 UInt8 数字。

## 语法

```sql
TO_MINUTE(<expr>)
```

## 参数

| 参数      | 描述       |
|-----------|------------|
| `<expr>`  | 时间戳     |

## 返回类型

 `TINYINT`

## 示例

```sql
SELECT
    to_minute('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ to_minute('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│                                      38 │
└─────────────────────────────────────────┘
```
