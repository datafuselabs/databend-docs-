---
title: SLEEP
---

在每个数据块上睡眠 `seconds` 秒。

!!! 警告 
    仅用于需要睡眠的测试场景。


## 语法

```sql
SLEEP(seconds)
```

## 参数

| 参数   | 描述 |
| ----------- | ----------- |
| seconds  | 必须是一个非负数或浮点数的常量列。｜

## 返回类型

UInt8

## 示例

```sql
SELECT sleep(2);
+----------+
| sleep(2) |
+----------+
|        0 |
+----------+
```
