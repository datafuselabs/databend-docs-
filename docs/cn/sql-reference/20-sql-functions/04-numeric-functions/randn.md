---
title: RAND(n)
---

返回一个在范围 `0 <= v < 1.0` 内的随机浮点值 v。要获取在范围 `i <= R < j` 内的随机整数 R，可以使用表达式 `FLOOR(i + RAND() * (j − i))`。参数 `n` 用作种子值。对于相同的参数值，RAND(n) 每次都会返回相同的值，从而生成可重复的列值序列。

## 语法

```sql
RAND( <n>)
```

## 示例

```sql
SELECT RAND(1);

┌────────────────────┐
│       rand(1)      │
├────────────────────┤
│ 0.7133693869548766 │
└────────────────────┘
```
