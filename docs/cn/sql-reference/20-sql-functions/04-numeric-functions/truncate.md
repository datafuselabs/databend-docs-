---
title: TRUNCATE
---

返回数字 `x`，截断到 `d` 个小数位。如果 `d` 为 0，结果没有小数点或小数部分。`d` 可以是负数，使得值 `x` 的小数点左边 `d` 位变为零。`d` 的最大绝对值为 30；超过 30（或 -30）的任何位数都会被截断。

## 语法

```sql
TRUNCATE( <x, d> )
```

## 示例

```sql
SELECT TRUNCATE(1.223, 1);

┌────────────────────┐
│ truncate(1.223, 1) │
├────────────────────┤
│ 1.2                │
└────────────────────┘
```
