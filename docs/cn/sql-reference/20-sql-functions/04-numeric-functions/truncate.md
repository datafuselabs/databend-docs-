---
title: TRUNCATE
---

返回数字 `x`，截断到 `d` 位小数。如果 `d` 为 0，则结果没有小数点或小数部分。`d` 可以为负数，使得 `x` 的小数点左侧的 `d` 位变为零。`d` 的最大绝对值为 30；超过 30（或 -30）的位数将被截断。

## 语法

```sql
TRUNCATE( <x, d> )
```

## 示例

```sql
SELECT TRUNCATE(1.223, 1);

┌────────────────────┐
│ truncate(1.223, 1) │
├────────────────────┤
│ 1.2                │
└────────────────────┘
```
